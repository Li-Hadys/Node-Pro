# Node Pro CLI

**Node Pro** 是一款专为网络中转（Forwarding）配置设计的自动化命令行工具。它通过核心的 **Tag 标签联动系统**，将复杂的落地节点信息与中转规则解耦，实现一键生成大规模中转订阅链接。



## ✨ 核心特性

- **多协议解析引擎**：深度集成核心解析器，支持从原始链接中提取协议（SS, VLESS, Hysteria2 等）、服务器、端口及 Query 参数。
- **Tag 联动映射**：通过标签系统实现“一对多”自动化映射。修改一个落地节点标签，所有关联的中转规则自动同步更新。
- **备注智能清洗**：内置正则清洗引擎，自动剔除旧的速率后缀（如 `-5Gbps`）和编号，按 `[名称]-[转发地] [递增数字]` 规范化重组。
- **全链路还原**：生成器支持保留原始链接的加密参数与查询字符串，确保中转后的链接 100% 可用。
- **ANSI 彩色交互**：纯 CLI 交互界面，适配 Linux 服务器及各平台终端，提供直观的状态反馈。

## 📂 项目结构

```text
project/
├── core/                # 核心逻辑模块
│   ├── parser.py        # 链接解析引擎 (Link -> Data)
│   ├── forward.py       # 中转映射与备注清洗算法
│   ├── generator.py     # 订阅链接还原生成 (Data -> Link)
│   └── storage.py       # 数据持久化管理
├── data/                # 本地数据库 (JSON)
│   ├── nodes.json       # 落地节点数据
│   └── forward_rules.json # 中转转发规则
└── main.py              # 程序入口 (交互式菜单)

```

## 🚀 快速开始

### 1. 环境准备

本项目采用 Python 原生库实现，**零第三方依赖**，克隆即可运行：

```bash
git clone https://github.com/Li-Hadys/Node-Pro.git
cd Node-Pro

```

### 2. 运行工具

```bash
python main.py

```

### 3. 使用流程

1. **导入节点**：选择菜单 `[1]`，批量粘贴你的落地节点原始链接（支持多行）。
2. **配置规则**：在 `data/forward_rules.json` 中定义中转机的 IP、端口和目标 Tag。
3. **生成链接**：运行生成功能，系统将自动匹配所有 Tag 并输出规范化的转发链接。
